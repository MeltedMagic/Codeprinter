# Codeprinter
Terminal application to convert code files to .tex files

## Requirements
* `pdflatex`
* `Pygments`

## How to Use
1. Specify absolute path to directory with python files you want converted to pdfs.
2. Edit Formatter for desired language
3. `python3 codeprinter.py`
